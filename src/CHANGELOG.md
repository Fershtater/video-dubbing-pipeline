# История изменений

Все значимые изменения проекта Video Dubbing Pipeline фиксируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
версионирование — [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Добавлено

- Асинхронная обработка TTS с ускорением в 3–5 раз
- Прогресс синтеза TTS в реальном времени
- Настраиваемая конкурентность (MAX_CONCURRENT)
- Асинхронный CLI (`cli_async.py`)
- Асинхронные функции TTS (`tts_async.py`)
- Асинхронное построение таймлайна (`timeline_async.py`)
- Документация по async API и примерам
- Лицензия MIT
- Рекомендации по контрибуции (CONTRIBUTING)

### Изменено

- Улучшен вывод в консоль и отображение прогресса
- Расширен Makefile целями для async
- Обновлена структура документации
- Документация перенесена в `docs/`

### Исправлено

- Проблемы с буферизацией вывода в Makefile
- Буферизация вывода Python (PYTHONUNBUFFERED=1)
- Отображение прогресс-бара в терминале

## [0.1.0] - 2025-09-19

### Добавлено

- Первый релиз
- Базовый конвейер дублирования видео
- Распознавание речи (faster-whisper)
- Синтез речи (OpenAI TTS)
- Полировка текста с GPT
- Построение таймлайна и выравнивание аудио
- Генерация и вжигание субтитров
- Описание и главы для YouTube
- Автоматизация через Makefile
- Управление зависимостями (Poetry)
- Документация

### Возможности

- **Извлечение аудио** — ffmpeg
- **STT** — локальная транскрипция (faster-whisper)
- **Полировка текста** — улучшение с помощью GPT
- **TTS** — OpenAI TTS с кэшированием
- **Таймлайн** — режимы segment, sentence, block
- **Субтитры** — генерация и встраивание SRT
- **YouTube** — описание, главы, теги
- **Детекция сцен** — границы сцен для блоков

### Технические детали

- Python 3.11+
- Poetry, FFmpeg, OpenAI API
- Модульная архитектура
- Обработка ошибок и кэширование TTS
